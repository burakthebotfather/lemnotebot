Notepad_bot - full project
-------------------------
Files included:
- src/bot.py           (main bot)
- src/db.py            (sqlite async helper)
- src/parser_utils.py  (triggers + chat map)
- requirements.txt
- Procfile
- Dockerfile (optional)
- .env.example

Instructions:
1. Fill TELEGRAM_TOKEN in environment or .env (Railway variables).
2. Install deps: pip install -r requirements.txt
3. Run: python src/bot.py
